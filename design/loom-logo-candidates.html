Loom · Logo 候选方案

三个方向，每个三种形态（主色版、纯色版、深色背景版）+ 真实场景预览。
挑一个方向深化，或者告诉我把谁的哪个元素搬到谁身上。
Direction A · 经纬 + 梭子 经典织机图像
简化的经纬交叉网格，加一个穿过的梭子。强隐喻——一眼就能看出"在织东西"。但小尺寸识别度偏低。
A1 · Main
三经一梭。梭子橙色穿过炭黑经线。
A2 · Dense
更密的经纬网格，梭子在下方穿过。
A3 · Dark mode
深色底，白色经线 + 橙色梭。
A4 · Mono
纯炭黑单色。印章、硬币可用。
Direction B · "L" 作为梭子 字母 + 织物一体
字母 "L" 的形态天然就像一把梭子。把 L 放进经纬里——既是首字母，又是织造工具。双重含义，小尺寸下也能认出是"L"。我最看好的方向。
B1 · Main
浅经线 + 粗 L 作为织出的形状。
B2 · L as weft
L 是穿过经线的一梭。
B3 · Woven L
L 是 3 经线 + 2 纬线织成的字形。
B4 · Dark
深色底版 B1。
Direction C · 纯编织纹 最极简抽象
几根经线几根纬线交错，没有具体物件——像 GitHub 编码图标那种抽象。小尺寸最清晰，但可能不够独特。
C1 · Grid
4 经 × 3 纬 的编织格。
C2 · Organic
纬线长度不同。暗示在织的过程。
C3 · Over-under
真正的上下交织效果。最诚实的"编织"。
C4 · Shuttle
三纬线 + 一个菱形梭子。
